## 注意事项

- 默认开学时间为"2022.02.28", 使用时请检查开学时间是否正确.
- 默认学期总周数为"18", 使用时请检查是否正确.
- 开学时间和默认学期总周数由于无法从网页提取, 所以初步计划为每学期适配.
